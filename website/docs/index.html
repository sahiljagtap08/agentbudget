Installation

pip install agentbudget

Requires Python 3.9+. No external dependencies — just the standard library.

For LangChain integration, install the optional dependency:

pip install agentbudget[langchain]

Quickstart

AgentBudget offers two modes: drop-in (zero code changes) and manual (explicit wrapping). Choose whichever fits your workflow.

Drop-in Mode (Recommended)

Add two lines to the top of your script. Every OpenAI and Anthropic call is tracked automatically.

import agentbudget
import openai

agentbudget.init("$5.00")

# Your existing code — no changes needed
client = openai.OpenAI()
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello"}]
)

print(agentbudget.spent())      # e.g. 0.0035
print(agentbudget.remaining())  # e.g. 4.9965
print(agentbudget.report())     # Full cost breakdown

agentbudget.teardown()  # Stop tracking, get final report

How it works: agentbudget.init() monkey-patches the Completions.create and Messages.create methods on OpenAI and Anthropic SDKs. When you call teardown(), the original methods are restored. This is the same pattern used by Sentry, Datadog, and other observability SDKs.

Drop-in API

FunctionDescription
agentbudget.init(budget)Start tracking. Patches OpenAI/Anthropic. Returns the session.
agentbudget.spent()Total dollars spent so far.
agentbudget.remaining()Dollars left in the budget.
agentbudget.report()Full cost breakdown as a dict.
agentbudget.track(result, cost, tool_name)Manually track a tool/API call cost.
agentbudget.get_session()Get the active session for advanced use.
agentbudget.teardown()Stop tracking, unpatch SDKs, return final report.

Manual Mode

For full control, use the AgentBudget class and BudgetSession context manager directly.

from agentbudget import AgentBudget

budget = AgentBudget(max_spend="$5.00")

with budget.session() as session:
    # Auto-cost LLM responses
    response = session.wrap(
        client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": "Analyze this..."}]
        )
    )

    # Track tool/API calls with known costs
    data = session.track(call_serp_api(query), cost=0.01, tool_name="serp")

    # Decorate functions with automatic cost tracking
    @session.track_tool(cost=0.02, tool_name="search")
    def my_search(query):
        return api.search(query)

print(session.report())
# {"session_id": "sess_...", "budget": 5.0, "total_spent": 3.42, ...}

Budget Envelope

A budget envelope is a dollar amount assigned to a unit of work — a session, a task, a user request, or a workflow run. Every cost incurred is tracked in real time. When the budget is exhausted, the session raises BudgetExhausted.

Budgets can be specified as strings or numbers:

# All of these work:
AgentBudget(max_spend="$5.00")
AgentBudget(max_spend="5.00")
AgentBudget(max_spend=5.0)
AgentBudget(max_spend=5)

Cost Sources

AgentBudget tracks three categories of cost:

  • LLM calls — Automatically costed using a built-in pricing table. Use session.wrap(response) or drop-in mode.
  • Tool calls — External APIs with known per-call costs. Use session.track(result, cost=0.01).
  • Decorated functions — Annotate any function with @session.track_tool(cost=0.02) to auto-track on every call.

Circuit Breaker

The circuit breaker provides three levels of protection:

Soft Limit (default: 90% spent)

Fires a warning callback when the session has used 90% of its budget. Your agent can use this to wrap up gracefully.

budget = AgentBudget(
    max_spend="$5.00",
    soft_limit=0.9,  # Warn at 90%
    on_soft_limit=lambda report: print("Warning: 90% budget used"),
)

Hard Limit (100% spent)

Raises BudgetExhausted immediately. No more calls allowed.

from agentbudget import BudgetExhausted

try:
    with budget.session() as session:
        while True:
            session.wrap(client.chat.completions.create(...))
except BudgetExhausted:
    print("Budget exhausted!")
    print(session.report())

Loop Detection

If the same tool or model is called N times within a time window, the circuit breaker trips early — even if budget remains. This catches infinite loops before they drain the budget.

budget = AgentBudget(
    max_spend="$5.00",
    max_repeated_calls=10,       # Trip after 10 repeated calls
    loop_window_seconds=60.0,    # Within a 60-second window
    on_loop_detected=lambda report: print("Loop detected!"),
)

When a loop is detected, LoopDetected is raised.

Async Support

AgentBudget supports async/await natively with AsyncBudgetSession.

from agentbudget import AgentBudget

budget = AgentBudget(max_spend="$5.00")

async with budget.async_session() as session:
    # Await and auto-cost in one call
    response = await session.wrap_async(
        client.chat.completions.acreate(
            model="gpt-4o",
            messages=[{"role": "user", "content": "Hello"}]
        )
    )

    # Sync wrap() still works for pre-awaited responses
    session.wrap(already_awaited_response)

    # track_tool works with both sync and async functions
    @session.track_tool(cost=0.01)
    async def async_search(query):
        return await api.search(query)

Nested Budgets

Parent sessions can allocate sub-budgets to child tasks. When the child finishes, its total spend is charged to the parent.

budget = AgentBudget(max_spend="$10.00")

with budget.session() as parent:
    # Allocate $2 for a sub-task
    child = parent.child_session(max_spend=2.0)
    with child:
        child.track("result", cost=1.50, tool_name="sub_task")

    # Parent is now charged $1.50
    print(parent.spent)      # 1.50
    print(parent.remaining)  # 8.50

The child budget is automatically capped at the lesser of max_spend and the parent's remaining balance.

Webhooks

Stream budget events to any HTTP endpoint. Useful for Slack alerts, Datadog, or billing systems.

budget = AgentBudget(
    max_spend="$5.00",
    webhook_url="https://your-app.com/api/budget-events",
)

Events are sent as JSON POST requests:

{
    "event_type": "soft_limit",  // or "hard_limit", "loop_detected"
    "session_id": "sess_abc123",
    "data": { ... }  // Full cost report
}

Webhook failures are logged but never raise — they won't break your agent.

Event Callbacks

Register Python callbacks for budget events:

budget = AgentBudget(
    max_spend="$5.00",
    on_soft_limit=lambda report: logger.warning(f"90% used: {report}"),
    on_hard_limit=lambda report: alert_ops_team(report),
    on_loop_detected=lambda report: logger.error(f"Loop: {report}"),
)

When webhook_url is also set, both your callback and the webhook fire.

LangChain Integration

pip install agentbudget[langchain]
from agentbudget.integrations.langchain import LangChainBudgetCallback

callback = LangChainBudgetCallback(budget="$5.00")

# Pass as a callback to any LangChain chain or agent
agent.run(
    "Research competitors in the CRM space",
    callbacks=[callback]
)

print(callback.get_report())

The callback tracks on_llm_end events automatically, extracting token usage and model name from LangChain's response objects.

CrewAI Integration

from agentbudget.integrations.crewai import CrewAIBudgetMiddleware

with CrewAIBudgetMiddleware(budget="$3.00") as middleware:
    # Use middleware.session to track costs
    result = middleware.track(
        crew.kickoff(),
        cost=0.50,
        tool_name="crew_run"
    )

print(middleware.get_report())
